# HephAuto

HephAuto is a modern automotive dealership website designed to provide a seamless experience for buying, selling, and exploring vehicles. The platform features a responsive design, interactive product galleries, advanced search and filtering, user authentication, and a professional page loader for enhanced user experience.

## Features

- **Hero Section:** Engaging video background with animated statistics and call-to-action.
- **Navigation:** Responsive navbar with dropdowns, search bar, and user actions.
- **Product Showcase:** Featured products, image galleries, and interactive product cards.
- **Mobile Support:** Mobile navigation, overlays, and bottom navigation bar.
- **User Authentication:** Login, signup, and profile management.
- **Checkout Flow:** Multi-step checkout process for purchases.
- **Admin Dashboard:** Manage listings, users, and site content.
- **Animations:** Smooth scroll animations and animated loaders.
- **Page Loader:** Three-dot animated loader on page entry/refresh.
- **Vercel Deployment:** Configured for easy deployment with custom routing and security headers.

## Project Structure

- `Heph.html` - Main homepage
- `Heph.js` - JavaScript for interactivity and dynamic features
- `Heph.css` - Stylesheet for layout, components, and animations
- `vercel.json` - Vercel configuration for routing and security

## Getting Started

1. **Clone the repository**
2. Open the project in Visual Studio Code
3. Use a local server (e.g., Live Server extension) to preview
4. Deploy to Vercel for production

## Deployment

The project is ready for deployment on Vercel. The `vercel.json` file handles routing to the homepage and sets security headers for best practices.
